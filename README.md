# Little Local

Little Local is a family-friendly place review website prototype designed to help parents and caregivers evaluate restaurants, attractions, and local businesses using practical criteria that traditional star ratings often overlook.

## Project Purpose

Parents often need specific information before choosing a place to visit with children. Little Local highlights details such as:

- Bathroom cleanliness
- Staff patience
- Wait time
- Stroller access
- Noise level
- Parking and safety
- Overall family value

This project demonstrates both front-end web development and the complete management of a small digital project.
